PHP환경구축

      php가 제대로 깔렷는지 확인.
    php파인을 만들고 아래 내용을 넣는다, 웹으로 열엇을때 php화면이 뜨면 정상이다
    <?php
    phpinfo();
    ?> 
    
    Bitnami개발자환경 설정
      c:\Bitnami\wampstack-8.0.7-1\php\php.ini,
    -display_errors = Off 를 display_errors = On 으로 변경.
    error Off이유: 우리한테 에러가 보인다는 것은 방문자한테도 에러 메세지가 보인다는 것이다.
    그 에러메시지엔 중요한 내용이 포함될수있다.디렉토리 경로,패스워드등.
    이것을 악용할수도 있기에 OFF한다. 
    나중에 실제로 서비스 할땐 꺼야한다.

    opcache.enable=1을 opcache.enable=0 으로 변경
    성능을 높이기 위한 설정 이다 
    켜져잇으면 php가 만들어 지는데 지연이 잇을수잇으니 꺼놓고 개발한느것이 좋다.
