배열 Array & 문법 Syntax

문자는 "따옴표로 시작해서 따옴표로 끝난다 "
배열은 [대괄호로 시작해서 대괄호로 끝난다. "그리고 이안에 값들을 적는데" "여러개의 값을 적을수잇다"["egoing","leezche"]등 "값과 값 사이는 ,콤마로 구분한다 " ]

var coworkers=["egoing","leezche"];

coworkers라는 변수에 이 ["egoing","leezche"] 배열이라고 하는 데이터 타입이 담긴것이다. 또한 데이터 타입에 coworkers라는 이름을 붙인거라 이해해도 된다

가져오기 get

document.write(coworkers[0]); egoing라는 값을 불로온다
document.write(coworkers[1]); leezche라는 값을 불러온다
get

배열 추가 add

coworkers.push('sung') coworkers.push('miki') .push를 사용하여 sung,miki라는 배열이 추가됨

개수확인 count

document.write(coworkers.length) length를 이용하여 coworkers라는 변수에 담겨잇는 배열개수를 구한다
