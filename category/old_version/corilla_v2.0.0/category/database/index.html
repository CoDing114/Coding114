자주 쓰는 명령어

    수정중 WHERE문을 빼먹으면 재앙이 올수잇다!!!
  • Create;생성
  • Drop;삭제
  • SHOW databases;데이터 베이스 확인
  • SUE;데이터 베이스 선택
  • DESC ;구조확인
  • SELECT * FROM topic;추가한 데이터 확인
  • ORDER BY id DESC;높은 값 부터 정렬
  • LIMIT 2;보이기 수량 설정가능
  • RENAME TABLE topic TO topic_backup;이름 변경
  • UPDATE topic SET description='Oracle is...',title='Oracle' WHERE id=2; 수정
  • DELETE FROM topic WHERE id=5; 데이터 삭제

자주 쓰는 용어

  • Table:표
  • Row,record:행
  • column:열
