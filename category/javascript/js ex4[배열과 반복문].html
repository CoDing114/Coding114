반복문 Loop & 배열 Array

배열에서 각각의 항목들을 원소(element)라 부른다

var coworkers = ['egoing','leezche','duru','mika','graphittie']

                
Co workers라는 배열에 잇는 원소들을 하나씩 불러와 나열한다

    ((+coworkers[i]+ 배열에서 데이터를 가져올땐 대괄호[]를 쓴다,
대괄호[] 안에는 가져오고자하는 데이터의 인덱스 값[i]을 넣는다))
<ul>
    <script>
        var i = 0;      -i라는 변수에 0이란 값을 넣어준다 
        while(i < 4){         -i의 값이 4보다 작을때 true
            document.write('<li>'+coworkers[i]+'</li>');
            i = i + 1;      -Loop 돌때마다 i값에 1씩 플러스 해준다
    }
    </script>
</ul>
            
데이터가 바뀐다구 로직이 바껴선 안된다

(( -length를 이용하여 배열안에 잇는 원소의 개수를 불러온다 ))

<ul>
    <script>
        var i = 0; 
            while(i < coworkers.length){
            document.write('<li>'+coworkers[i]+'</li>');
            i = i + 1; 
        }
    </script>
</ul>
            
                    
리스트에 링크 삽입

      ((<a href="http://baidu.com/'+coworkers[i]+'">를 추가하여
