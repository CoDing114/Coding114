pip와 pip3 차이

pip : python2 버전
pip3: python3 버전

설치


                    pip install some-package-name

설치list


                    pip list

패키지 정보


                    pip show some-package-name

제거


                    pip uninstall some-package-name

업그레이드


                    pip install --upgrade some-package-name

특정 버전의 (또는 contrib 버전) opencv 패키지 설치


                    pip install some-package-name==3.4.5.20

함수

함수를 정의할땐 def로 시작하며 함수 이름을 적고, 괄호() 열고 닫고 나서, 콜론":"으로 정의 하고나서 함수내에서 실제로 수행되는 내용을 작성 하면된다

함수 생성후엔 호출해줘야 실행 될수잇다


  def open_account(): # 함수 정의
      print("새로운 계좌가 생성되었습니다")

  open_account() # 함수 호출

긴코드 줄바꿈

긴코드 사용중엔 "\"를 넣으면 코드에 영향없이 보기좋게 줄바꿈하여 사용할수잇다. 

  def profile(name, age=20, main_lang="python"):
      print("이름:{0}\t나이:{1}\t언어:{2}" \
        .format(name, age, main_lang))


  profile("CoRilla")
  profile("MIKI", 20, "Java")

# 4.5 탈출 문자


  # \n :줄 바꿈
  print("문장 \n 줄 바꿈")

  # \": 문장내 \" \' 따옴표
  print("저는 '코릴라' 입니다")
  print("저는 \"코릴라\" 입니다")

  # \\: 문장내 역슬러시 \
  print("C:\\Bitnami\\wampstack-8.0.8-0\\apache2\\htdocs> ")  # 역슬러시 두개를 하나로 간주

  # \r: 커서 맨 앞으로 이동
  print("Red Apple\rPine")

  # \b: 백스페이스 (한글자 삭제)
  print("Redd\b Aapple")

  # \t : Tap 탭
  print("Red\tApple")

print할때 뒤에 이어 프린터 하고 싶을때 ,end""활용


  def profile(name, age, *languages):
  print("이름:{0}\t나이:{1}\t".format(name, age), end="언어:")
  for lang in languages:
      print(lang, end=" ")
  print()


  profile("Corilla", 20, "python", "java", "C", "C++", "C#")
  profile("MIKI", 20, "python", "Java")

    {
      "version": "0.2.0",
      "configurations": [
          {

              "type": "python",
              "request": "launch",
              "program": "${file}",
              "console": "integratedTerminal",
              "cwd": "${fileDirname}" //  "Current File" 
          },
      ]
  }
